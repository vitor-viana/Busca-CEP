# Busca-CEP

Este projeto tem como finalidade, buscar endereços através de CEPS fornecidos pelo usuário.
Para este projeto utilizei a API viacep.com.br/ws/"CEP"/json, junto com:
 - ReactJs
 - Axios
 - React Icons
